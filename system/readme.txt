						软件使用说明书
1. 系统配置：本软件要求运行在Windows 7及以上版本操作系统的机器上，且要求机器配置java 环境，jre1.7 及以上版本都可以。运行时请确保屏幕分辨率为 1920 x 1080p 及以上，系统内存至少为8G，否则将无法很好地显示仿真演示效果
2. 文件结构说明：MyProject为Java工程文件
		 data为仿真输入/输出数据
                 可执行程序为演示结果
	
3 软件运行：首先将data文件夹放入C盘根目录
	    双击ran.bat即可运行程序
	    进入开始界面点击“Begin”后进入仿真显示界面
	    点击“Run”开始仿真
	    点击“exit”可结束仿真
